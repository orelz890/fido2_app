
## Init Server App
```bash
mkdir fido2-auth-server
cd fido2-auth-server
npm init -y
```

### Setup:
```bash
npm install express body-parser fido2-lib cors base64url express-session
```

### Start Backend:
```bash
node server.js
```

## Init Client React App
```bash
npx create-react-app fido2-auth-client
cd fido2-auth-client
```

### Setup:
```bash
npm install axios
```
### Start Frontend:

```bash
npm start
```
